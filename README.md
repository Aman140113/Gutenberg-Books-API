### API Usage Examples

##  Basic search
```
GET /books?limit=10
```
## Filter by multiple criteria
```
GET /books?language=en&topic=children&author=carroll
```
## Multiple values per filter
```
GET /books?language=en,fr&topic=child,infant
```
## Pagination
```
GET /books?limit=25&offset=50
```
